KindleSuite 3.5

KindleSuite is a tool suite designed to collect all of the tools available for developing and modifying the Amazon Kindle Fire HD (7") 2013 into one User Interface to make things as easy as possible for endusers.

KindleSuite includes the following features:

  • Backup & Restore Apps
  • Factory Restore
  • Root
  • Install Gapps
  • Install ROM's

You can find guides in our wiki
