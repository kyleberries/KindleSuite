KindleSuite

KS4.0 is currently under development. For a working earlier version see link below...

KindleSuite is a tool suite designed to collect all of the tools available for developing and modifying the 2013 Amazon Kindle Fire HD (7").

KindleSuite includes the following features:

Root

Factory Restore

Backup&Restore /Data

Install ROMs

Install Gapps
