KindleSuite is a tool suite designed to collect all of the tools available for developing and modifying the 2013 Amazon Kindle Fire HD (7") Into one User Interface to make things as easy as possible for power modders and new users.

KindleSuite includes the following features:

  • Root
  • Factory Restore
  • Backup&Restore /Data
  • Install ROMs
  • Install Gapps
